### 오늘 배운 것
- 자바의 조건문
- if 문
```java
if(조건){
    실행문
}else if(조건2){
    실행문2
}else{
    실행문3
}
```
- switch문
```java
switch(식){
    case 값1:
        실행문장1
    break ;
    case 값2:
        실행문장2
    break ;
    default:
    break ;
}
```

<br/>
<br/>
<br/>

### 어려웠던 점, 생각할 점
- switch 문은 break을 안 쓴다면 첫 조건이 만족해도 끝까지 실행된다. 오히려 이걸 이용해서 코드를 짤 수 있지 않을까?
- if 문이나 switch문 중 어떤 상황에 뭐를 선택하는게 효율적인지 고민해봐야겠다.
- 깔끔하게 코드를 짜기가 어렵다. 코드를 조금 더 단축시키고 가독성을 높일 방법을 찾아야겠다
