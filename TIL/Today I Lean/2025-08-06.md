## TIL 2025-08-06

### 오늘 배운 것
- Spring 스크립트릿, 표현식

<br/>

- 스크립트릿
```jsp
<%
int num = 10;
%>
```
jsp에서 자바의 영역을 만들 수 있음

<br/>

- 표현식
```jsp
<%= num %>
```
자바 변수나 메소드의 결과값을 웹페이지에 표현할때

<br/>
<br/>
<br/>

### 어려웠던 점, 생각할 점
- 스크립트릿이나 표현식은 for문안에서도 사용 할 수 있다. 이걸 이용해 여러 가지를 활용할 수 있을거 같다

<br/>
<br/>
<br/>

### 개선 계획
- 스크립트릿을 활용해서 웹에 여러 기능을 넣어보자

<br/>
<br/>
<br/>

### 느낌점
- for문이나 if문 안에서도 스크립트릿을 쓸 수 있다는 걸 바로 떠올리지 못 했다 조금더 유연한 사고가 필요할거 같다
