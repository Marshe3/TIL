## TIL 2025-09-15

### 오늘 배운 것

리눅스 명령어
root@myapp-web:~#
root -> 접속 계정
@myapp-web -> 컴퓨터 주소
~ -> 현재 디렉토리 경로
# -> 관리자 계정 / $ -> 일반사용자 계정
cd 디렉토리명 : 폴더를 이동하고 싶을때 사용
ls [옵션] : 폴더 안에 존재하는 파일 or 폴더들을 확인
-l (상세정보) -a(숨김폴더까지 확인)
pwd: 현재 내가 작업중인 디렉토리 경로 확인
mkdir 폴더명 : 폴더 생성 가능
rm 파일명 or 폴더명 : 폴더를 삭제할 때는 -r 넣어야한다
cp 파일명 or 폴더명
mv 파일,폴더 경로: 파일 이동
mv 파일명 새로운 파일명: 이름을 변경
touch 파일명: 새로운 파일을 생성
vim 파일명 : 편집기를 사용해서 파일 수정가능
* vim 편집기 많이 사용! 단축키를 외워주기
cat 파일명 : 해당하는 파일 내용을 간단하게 출력
cat >> 파일명: 해당하는 파일 안쪽에 새로운 내용 추가
cat > 파일명: 기존에 있던 내용은 지우고 새로운 내용 추가
* 나가고 싶을 때, ctrl d
chmod 권한 : 권한 변경 (문자, 8진수 숫자)
r (4) w(2) x(1) -> 7 (rwx)
chown 소유자 파일명 : 해당 파일의 소유자 변경
* ubuntu 기준 -> 패키지 설치 : apt (advanced package tools)
(1) 잘 작동이 안된다? apt update
(2) 패키지 설치 : apt install 패키지명 -y
* sudo(super user do) -> 임시 관리자 권한을 빌리는 개념
* 우리는 어차피 root 계정을 사용할 예정이기에 쓸일이 x
* 설치한 패키지의 상태를 확인 실행
systemctl start 패키지명
systemctl status 패키지명
systemctl restart 패키지명
systemctl stop 패키지명
